# SolarPro: природне відео без затемнення

## Мета

Показувати `1.mp4` з його власною яскравістю та вбудованим переходом «день → ніч». Поверх відео залишити лише маркетинговий текст, переваги, меню та дві CTA-кнопки.

## Погоджений дизайн

- Повністю прибрати постійний локальний градієнт зліва та знизу.
- Повністю прибрати загальний синьо-графітовий нічний шар.
- Прибрати CSS-змінну `--scene-darkness` і будь-яку синхронізацію затемнення з часом відео.
- Не застосовувати до відео `filter`, `opacity`, маски або інші корекції кольору.
- Залишити білий текст, легку тінь лише безпосередньо біля літер, переваги, кнопки та меню.
- Зберегти природний перехід день → ніч, який уже міститься у відеофайлі.

## Розглянуті варіанти

1. **Відео без будь-якого оверлею — обрано користувачем.** Максимально точно показує оригінальний ролик.
2. **Локальний градієнт лише під текстом — відхилено.** Навіть він робить початковий кадр надто синім.
3. **Напівпрозора картка під текстом — відхилено.** Суперечить вимозі залишити поверх ролика просто текст.

## Адаптивність

- На комп’ютері, планшеті й телефоні оригінальна яскравість відео залишається однаковою.
- Поточні розміри, переноси тексту, положення відео та CTA не змінюються.
- На мобільному немає окремого затемнення; текстова тінь може залишатися для локальної читабельності літер.
- Не допускаються горизонтальне прокручування, обрізання кнопок або зміна `object-fit: cover`.

## Ізоляція

- Змінити `video-fullscreen.html` і лише fullscreen-секцію `styles.css`.
- Видалити підключення `fullscreen-video-sync.js` та сам невикористовуваний скрипт разом із його окремим тестом.
- Оновити `tests/video-fullscreen.contract.ps1`, щоб він вимагав повної відсутності оверлеїв.
- Не змінювати `photo-benefits.html`, `video-variant.html`, тексти, кнопки або відеофайл.

## Перевірка

- Спочатку контракт має не пройти на поточній сторінці через наявні `::before`, `::after`, `--scene-darkness` і підключений скрипт.
- Після реалізації контракт підтверджує відсутність усіх чотирьох механізмів затемнення.
- Усі попередні контракти мають залишитися зеленими.
- Локальний сервер має повертати оновлені HTML і CSS зі статусом 200.

## Критерій готовності

Повноекранна сторінка відтворює відео без доданого затемнення або кольорової корекції, а поверх нього залишаються тільки наявні елементи інтерфейсу й текст.
